# Anti-Slop: Typography

- **The Slop**: Menggunakan font Inter/Roboto secara default untuk semua hal, dengan bobot font yang membosankan dan warna abu-abu generik.
- **The Fix**: Gunakan font Serif berkualitas untuk judul (Display), atau buat tracking (letter-spacing) sangat rapat pada heading besar (-0.02em hingga -0.04em).
