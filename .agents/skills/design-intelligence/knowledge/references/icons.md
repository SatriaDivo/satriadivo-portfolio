# Iconography Reference (shadcn)

- **Standard**: Gunakan library konsisten seperti Lucide (yang digunakan shadcn/ui).
- **Rule**: Ketebalan stroke ikon (stroke-width) harus konsisten di seluruh aplikasi (biasanya 1.5px atau 2px).
- **Anti-pattern**: DILARANG mencampur gaya ikon (outline dengan solid) atau menggunakan emoji OS sebagai pengganti ikon profesional.
