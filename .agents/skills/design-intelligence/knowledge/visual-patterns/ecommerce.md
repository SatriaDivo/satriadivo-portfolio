# E-commerce Aesthetic Pattern

- **Characteristics**: Product is king. UI harus tidak terlihat (invisible UI) agar produk menonjol.
- **Layout**: Grid yang rapat tapi sangat rapi untuk katalog. Halaman detail produk yang luas.
- **Anti-pattern**: UI yang terlalu mendominasi visual (warna background mencolok) mengalahkan warna produk itu sendiri.
