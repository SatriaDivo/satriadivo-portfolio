# Fintech Aesthetic Pattern

- **Characteristics**: Trustworthy, clean, data-heavy but scannable. Dominan warna putih/gelap solid dengan aksen biru/hijau terpercaya.
- **Typography**: Gunakan monospace atau font geometrik untuk angka/saldo agar sejajar rapi.
- **Anti-pattern**: Hindari ilustrasi kartun kekanak-kanakan atau gradasi warna berlebihan.
